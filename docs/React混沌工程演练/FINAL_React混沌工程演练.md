## 🚀 使用指南

### 快速开始

```bash
# 1. 安装依赖
npm install

# 2. 启动开发服务器
npm run dev

# 3. 查看故障类型
npm run chaos list

# 4. 注入故障
npm run chaos inject --type component-crash

# 5. 提交并推送
git add .
git commit -m "test: inject fault"
git push origin main

# 6. 恢复正常
npm run chaos restore
```

### 部署到Vercel

```bash
# 1. 安装Vercel CLI
npm i -g vercel

# 2. 登录Vercel
vercel login

# 3. 部署
vercel

# 4. 配置GitHub Webhook
# 在GitHub仓库设置中添加Webhook
# Payload URL: https://your-fault-analysis-platform.com/webhook
# Content type: application/json
# Events: push, deployment_status
```
